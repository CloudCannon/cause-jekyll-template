Our Mission

The JVM offers a lot of capabilities: make network calls, compile code on the fly, run code on the fly, etc. Your application probably doesn't need all of this. Hence, for an attacker, they just represent a welcome attack surface. The Security Manager is the way to apply the least privilege principle to reduce this attack surface via a configuration file. JEP 411 aims to remove the Security Manager with no replacement. Don't be a sheep, fight against the JEP!
